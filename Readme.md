# Escape Tales

## Project Overview
*Escape Tales* is an immersive project centered around the concept of storytelling through escape-room-style puzzles. Combining narrative depth with challenging problem-solving elements, the project aims to engage participants in an interactive and memorable experience. Each tale unravels as participants solve puzzles, uncover secrets, and make choices that influence the story's outcome.

## Features
- **Dynamic Storylines**: Multiple tales with branching narratives based on player decisions.
- **Challenging Puzzles**: A variety of logic, math, and lateral-thinking challenges integrated into the stories.
- **Interactive Gameplay**: Use of digital or physical elements to enhance engagement.
- **Replayability**: Different endings based on player choices and puzzle-solving paths.
- **Multiplayer Option**: Collaborative or competitive modes for groups.

## Goals
1. **Create Immersive Experiences**: Blend storytelling and puzzles seamlessly.
2. **Foster Collaboration**: Encourage teamwork through group play options.
3. **Appeal to a Broad Audience**: Make the project accessible and enjoyable for both casual and experience players.

## Technologies
- **Frontend** :- HTML, CSS (for web designing and all)
- **Backend**: Node.js, Express.js, and JavaScript (for web-based experiences).
- **Database**: Firebase or MongoDB (for saving player progress and choices).
- **Hosting**: Vercel (for hosting).
